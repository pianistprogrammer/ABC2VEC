================================================================================
                    ABC2VEC PACKAGE - INSTALLATION GUIDE
================================================================================

CONGRATULATIONS! You have a production-ready Python package extracted from
your Jupyter notebooks.

--------------------------------------------------------------------------------
QUICK INSTALL (Recommended)
--------------------------------------------------------------------------------

1. Navigate to package directory:
   $ cd abc2vec_package

2. Run automated setup:
   $ ./setup.sh

3. Verify installation:
   $ python verify_installation.py

That's it! You're ready to go.

--------------------------------------------------------------------------------
MANUAL INSTALL
--------------------------------------------------------------------------------

1. Create virtual environment:
   $ python -m venv venv
   $ source venv/bin/activate    # Windows: venv\Scripts\activate

2. Install package:
   $ pip install -e .

3. Verify:
   $ python verify_installation.py

--------------------------------------------------------------------------------
MAKEFILE COMMANDS
--------------------------------------------------------------------------------

$ make install      # Install package
$ make verify       # Verify installation
$ make test         # Run tests
$ make format       # Format code
$ make data         # Process dataset
$ make train        # Train model (quick)
$ make help         # Show all commands

--------------------------------------------------------------------------------
FIRST RUN
--------------------------------------------------------------------------------

After installation:

1. Process Data (5-10 minutes):
   $ python scripts/run_data_pipeline.py

2. Train Model (quick test - 5 minutes):
   $ python scripts/run_training.py --epochs 2 --batch_size 32 --d_model 128

3. Run Example:
   $ python scripts/example_usage.py --vocab_path ./data/processed/vocab.json

--------------------------------------------------------------------------------

# Encode ABC tune

patches = patchifier.patchify(tune)

TROUBLESHOOTING
--------------------------------------------------------------------------------

Problem: Import errors
Fix:     $ pip install -e .

Problem: Missing dependencies
Fix:     $ pip install -r requirements.txt

Problem: CUDA out of memory
Fix:     Use smaller batch size: --batch_size 16

Problem: Tests failing
Fix:     $ pip install -e ".[dev]" && pytest tests/ -v
